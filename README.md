# YAML Configuration and Ansible Project

## Overview

This project demonstrates the use of YAML configuration files and a basic Ansible playbook for automation and configuration management.

## Files

### config.yaml

Stores application settings and file categorization rules using YAML syntax.

### install_nginx.yml

An Ansible playbook that automates the installation of Nginx on Linux systems.
